# Python

El presente proyecto forma parte del curso de Python y tiene como objetivo aplicar los conocimientos adquiridos a través de la resolución de distintas consultas o katas. Estas actividades permiten poner en práctica las principales funcionalidades del lenguaje, utilizando diferentes enfoques para llegar a un mismo resultado, lo que fomenta la comprensión profunda de la sintaxis y las estructuras de Python.

A lo largo del proyecto se demuestra el manejo de tipos de datos básicos, el uso de estructuras de datos y sus métodos, la implementación de condicionales y estructuras de iteración, así como la creación y utilización de funciones y clases dentro del paradigma de programación orientada a objetos. Además, se incorporan módulos y se aplican buenas prácticas de programación, asegurando que el código sea legible, modular y eficiente.

En conjunto, este trabajo busca consolidar las competencias fundamentales del lenguaje Python mediante la resolución práctica de ejercicios que simulan problemas reales de programación.
